# Tableau Cheat Sheet: Layout & Design

### 1. Show/Hide Buttons
Create "pop-out" menus for filters or instructions to save space.
1.  Select a **Floating Container**.
2.  Click the container's dropdown menu and select **Add Show/Hide Button**.
3.  Edit the button to use a custom image (like a hamburger menu icon or an "i" for info).

### 2. The "Blank" Spacer Trick
*   **The Tip:** Use the **Blank** object inside containers to force charts to specific widths or to create "padding" that is more reliable than the standard padding settings.

### 3. Transparent Shapes
*   **The Tip:** Import a 1x1 pixel transparent PNG into your **Shapes** folder. Use this for "Navigation" buttons or to create invisible triggers that don't distract from the visual design.

### 4. Custom Color Palettes
*   **The Trick:** Edit the `Preferences.tps` file in your **My Tableau Repository** folder to add your brand's specific Hex codes as a custom palette option.
