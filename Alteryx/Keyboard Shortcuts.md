# Alteryx — Essential Keyboard Shortcuts

### Workflow Control
*   **Ctrl + R**: Run the workflow (or Stop it).
*   **Ctrl + S**: Save the current workflow.
*   **Ctrl + Shift + B**: Add a **Browse tool** to all output anchors of the selected tool(s).
*   **F1**: Open the help page for the selected tool.

### Canvas Navigation
*   **Spacebar + Left Click**: Pan around the canvas.
*   **Ctrl + Mouse Scroll**: Zoom in and out.
*   **Ctrl + 0**: Zoom to see the entire workflow at once.
*   **Ctrl + Tab**: Switch between multiple open workflow tabs.

### Editing Tools
*   **Ctrl + F**: Open the **Find** tool to search for specific text or tool numbers.
*   **Shift + Delete**: Delete a tool and **automatically connect** the upstream and downstream tools around the gap.
*   **F2**: Quickly open the **Annotation** window for the selected tool.
*   **Ctrl + Z / Y**: Standard Undo and Redo.
