## 1. Adding New Python Packages to Server
Used for installing packages on a server that may not have direct internet access.

#### Download package on laptop
```bash
# Downloads the package and its dependencies to a specific folder
pip download package_name -d "directory_path"
```

#### Install package on server
```bash
# Installs from the local directory without searching the online index
pip install package_name --no-index -f "directory_path"
```

---

## 2. Multiple Versions of Python
Managing packages when you have several Python versions (e.g., 3.10, 3.12) installed.

#### View package list
```bash
# Lists all installed packages for a specific Python version
py -3.12 -m pip list
```

#### Install specific package version
```bash
# Installs a precise version of a library to a specific Python environment
py -3.12 -m pip install selenium==3.0.0
```
