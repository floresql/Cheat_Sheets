# 🐍 Python Basics Cheat Sheet

## ✍️ Syntax & Variables
```python
x = 5
name = "Alice"
PI = 3.14
```

## 🧮 Arithmetic Operators
```python
+  -  *  /  //  %  **
```

## 🔁 Conditionals
```python
if x > 0:
    print("Positive")
elif x == 0:
    print("Zero")
else:
    print("Negative")
```

## 🔁 Loops
```python
for i in range(5):
    print(i)

while x < 10:
    x += 1
```

## 🧰 Functions
```python
def greet(name):
    return f"Hello, {name}!"
```

## 🧠 List Comprehensions
```python
squares = [x**2 for x in range(10)]
```

## 🔎 Useful Built-ins
```python
len(), type(), int(), str(), float(), dir(), help()
```
